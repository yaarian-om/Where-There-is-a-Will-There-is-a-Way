Just Follow the codes as written below depends on the current situation you have.

###### First Time / One time use
1. Download Git Repository
	|- git clone write_The_Url_of_the_Repository
2. Create New Branch in your PC
	|- git checkout -b new_Branch_Name_Here
	|- git branch 
		|- After pressing enter the name of the new branch should be highlighted

###### Every Time
1. Before writing your codes
	|- git pull
2. After Writing all your codes and updates
	|- git add .
	|- git commit -m "Some message about updates"
	|- git push --set-upstream origin new_Branch_Name_Here
